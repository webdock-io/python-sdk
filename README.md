# Python SDK for Webdock API
This is the Python SDK to interact with Webdock API.

## Installation
Install from PyPi:
```shell
pip install webdock
```
or clone this repo and run:
```python
python setup.py install
```
### Usage
```python
from webdock.webdock import Webdock

wd = Webdock('your-api-key-here')
```
